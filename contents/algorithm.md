### Dynamic Programming이란? 장점은?

 피보나치를 통한 재귀와 DP 비교 설명

```
N = int(input())
D = [0, 1]

for i in range(2, N + 1):
    D.append(D[i-2] + D[i-1])

print(D[N])
```
