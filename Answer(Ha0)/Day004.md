# Day 004

__Question__ 다음 용어를 설명하세요.
```
(a) DPI
(b) Firewall
(c) MTU
(d) Segment
(e) 망중립성
```

__Answer__
```
(a) DPI
- Deep Packet Inspection의 약어로, 인터넷을 통해 전송 된 패킷에 대한 심층 분석에 사용된다. 

(b) Firewall: 
- 외부 네트워크로부터 내부 네트워크를 보호하기 위해 네트워크 트래픽을 모니터링하고 제어하는 네트워크 보안 시스템이다.

(c) MTU 
- Maximum Transmission Unit의 약어로 어떤 네트워크에서 하나의 패킷 또는 프레임에 담아 전송될 수 있는 최대 크기를 의미한다.

(d) Segment
- 네트워크 케이블과 같이 물리적으로 분리되는 네트워크 구분 단위이다. 

(e) 망중립성
- 모든 네트워크 사업자들이 인터넷에 존재하는 서비스나 컨텐츠들을 동등하게 취급하고, 비차별, 상호접속, 접근성의 원칙을 동일하게 적용해야 한다는 조건이다. 
```
